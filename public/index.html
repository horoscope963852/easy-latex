    First Login Guide
    
        
    Welcome to easy-latex
    
        
    This quick guide introduces the main features and the recommended workflow.
    
        
      
          
    1. Open the top Project menu to switch, create, rename, or delete projects.
      2. 
          
    2. Open the Files menu to upload `.tex` files, create new files, and choose the document you want to edit.
      3. 
          
    3. Open the Build menu to change the engine, select the main `.tex` file, inspect compile logs, and enable auto compile.
      4. 
          
    4. Use Compile to refresh the right-side PDF preview, then Export PDF to download or save the result.
      5. 
          
    5. Shortcuts: `Ctrl/Cmd + S` saves, `Ctrl/Cmd + Enter` compiles, `Ctrl/Cmd + Shift + E` exports, and `Alt + 1/2/3` opens the top menus.
      6. 
        
    
        
    For Chinese documents, XeLaTeX is the default and is usually the right choice.
