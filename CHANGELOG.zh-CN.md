# 更新日志

本文件记录项目的重要变更。

项目采用语义化版本号（`MAJOR.MINOR.PATCH`），Git 标签格式为 `vX.Y.Z`。

## [1.0.0] - 2026-03-31

### 新增

- `easy-latex` 首个公开版本
- 单机部署的在线 LaTeX 编辑与编译服务，支持源码/PDF 分栏视图
- 管理员控制的用户名密码用户系统，以及可选游客模式
- 支持多文件 LaTeX 项目，并可手动选择主 `.tex`
- PDF 导出、用户空间配额、游客清理机制
- 中英双语界面切换与首次登录引导
- 面向 XeLaTeX 的默认配置，以及中文/CJK 诊断提示
- 一键安装脚本、systemd 单元文件与验证脚本
- 用于容器化复刻的 Docker 部署文件
