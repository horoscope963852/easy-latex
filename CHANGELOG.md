# Changelog

All notable changes to this project should be documented in this file.

The project follows Semantic Versioning (`MAJOR.MINOR.PATCH`) and Git tags in the form `vX.Y.Z`.

## [1.0.0] - 2026-03-31

### Added

- Initial public release of `easy-latex`
- Single-host web editor and compiler with source/PDF split view
- Admin-managed username/password accounts and optional guest mode
- Multi-file LaTeX projects with selectable main `.tex`
- PDF export, per-user quota, and guest cleanup
- Chinese/English UI toggle and onboarding guide
- XeLaTeX-oriented defaults and CJK diagnostics
- Bootstrap scripts, systemd units, and verification tooling
- Docker deployment files for containerized reproduction
